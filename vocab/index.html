EBV Phenology Vocabulary

Temporary GitHub Pages vocabulary for the EBV Phenology Building Block. These term IRIs are used by the OGC Building Blocks viewer while the vocabulary is not yet registered in OGC Rainbow.

Base URI: https://maytetoscano.github.io/bblock-ebv-bioclima/vocab/#

Terms

  • entity_type — Forest Entity Type
  • year — Observation Year
  • vap_doy — Start of VAP (Day of Year)
  • mean_doy — Mean DOY (spatial average)
  • min_doy — Minimum DOY
  • max_doy — Maximum DOY
  • stddev — Standard Deviation of DOY
  • valid_percent — Valid Pixel Percentage
  • crs — Coordinate Reference System
  • grid_width — Grid Width (columns)
  • grid_height — Grid Height (rows)
  • sensor — Source Sensor Platform
  • algorithm — Derivation Algorithm
  • trend_slope — Trend Slope (days/year)
  • anomaly — Annual Anomaly Status
  • ConiferousForest — Coniferous Forest
  • DeciduousForest — Deciduous Forest

Forest Entity Type

https://maytetoscano.github.io/bblock-ebv-bioclima/vocab/#entity_type

Type
Object property
Label — EN
Forest Entity Type
Label — ES
Tipo de entidad forestal

Definition: The ecosystem entity type for which the phenology metric is derived. Values may identify forest entities such as coniferous or deciduous forest.

Definición: Tipo de entidad ecosistémica para la que se deriva la métrica fenológica. Puede identificar entidades forestales como bosque de coníferas o bosque caducifolio.

Range: IRI or string
